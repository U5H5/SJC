# SCJ - Simple Junk Code
Simple C++ tool for adding junk code to your program. 

## Requirements

* Built with **MSBuild** on **Visual Studio 2022**
* Uses **C++20**
* Junk code is defined in `junk.h`, you may have to change seed.
